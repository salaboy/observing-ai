# OpenTelemetry Observation Experiments

This repository contains a set of experiments related with observing different coding agents, agentic frameworks and tools in the AI/ML space using OpenTelemetry. 

The objective is to build a collection of quick (but more advanced than "hello worlds") to validate where the observability space is moving in regards with the use of semantic conventions, available telemetry and reactive capabilities. 

## Projects

Projects are divided by programming language stack or coding agent harnessess.
